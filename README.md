##구현시 어려웠던 점
- 눌렀을 때 크기를 작게 하는데 겉만 생각하여 클씨 크기를 고려하지 않아 글씨는 작아지지 않고 잘려 그것을 고려하여 다시 실행했어야 했던 점이 쉽지만은 않았던 것 같다. 또한 메시지를 출력하거나 다시하기를 만들려면 버튼을 또 추가해야될 줄 알았는데 코드만으로도 구현이 가능하다는 것을 알게 되었다.
